# Generate & export a report

_Produce a point-in-time report for a review, auditor, or customer._

Reports are scope-aware — set the sidebar customer scope first if you want a per-customer report rather than a fleet-wide one.

## Steps

1. Set the customer scope in the sidebar (or leave All Customers for a fleet report).

2. Open the Reports hub and pick a generator (Posture, Compliance, Security, SLA, Cost & Margin, Incident, DSAR, or Customer Account).

3. Review the RAG status banner and the report body.

4. Export with Download CSV or Download Markdown, or use Print / Save as PDF for the formatted version.

Tip: many pages can launch their own report directly (e.g. Generate report on Cost & Margin).
