# Edit on-call & rotation

_Reassign the on-call responders and advance the rotation._

The Primary responder drives every routed signal, so keep on-call current — you can edit it directly from the Notifications page.

## Steps

1. Open Notifications and find the On-Call card.

2. Click Edit and reassign Primary, Secondary, and Manager from the team roster.

3. Save — the “On-call now” tile and PagerDuty routing update immediately.

4. To advance the schedule, edit the Upcoming Rotation and use “Make current” to apply a week to the live assignment.

5. Verify the change by checking who the router now shows as on-call.
